Enumerable Methods

The Power of group_by

May 26, 2015

Enumerable is a mixin in Ruby that provides tons of great methods that allow you to search and sort data. One of the enumerable methods that I'm sure you've seen is sort, and we all know it is a lifesaver. I'm going to tell you about the group_by method, which is amazing if you know how to use it.

Essentially, group_by groups a collection of data based on an argument, and that argument is a block. The output will be a hash where the keys are the results from the block and the values are arrays containing elements in the initial collection that correspond to each key. Let me show you.

Lets make an array. How about we include some of our favorite foods?

foods = ["chicken", "spaghetti", "potatoes", "beef", "fries", "beans", "cake", "wings", "roscoes"]

Now, lets group each of these foods by length.

foods_by_length = foods.group_by {|food| food.length}

What happens when we run this?

{7=>["chicken", "roscoes"], 9=>["spaghetti"], 8=>["potatoes"], 4=>["beef", "cake"], 5=>["fries", "beans", "wings"]}

Look at that! We got a new hash where the keys are the lengths of the food items and the values are arrays of each food item that has the same length as the value! You can do with this information what you wish, now go forth and group_by, young padawan.
